# 멘탈 한마디 카테고리 리서치

작성일: 2026-06-07
상태: provisional
방법: Superpowers brainstorming + gstack office-hours 제품 리서치 관점 + 공개 웹/커뮤니티 검색

정정: 2026-06-08 기준 최종 앱명은 `칭찬해줘`이다. 이 문서의 `멘탈 한마디`는 당시
이름 후보이자 카테고리 프레임으로 남긴다.

## 질문

`멘탈 한마디` 앱에서 5개 버튼을 어떤 카테고리로 잡아야 하는가?

기존 후보는 다음이었다.

- 힘내자
- 진정하자
- 쉬어도 돼
- 괜찮아
- 알아줘/칭찬해줘

사용자 피드백:

- `잘하고 있어`에서 갑자기 `알아줘`로 바뀐 이유가 약하다.
- 실제 사람들이 듣고 싶어 하는 말의 종류를 Instagram/커뮤니티 기반으로 다시 확인해야 한다.

## 리서치 한계

Instagram의 실제 태그 페이지는 공개 검색/크롤링에서 직접 접근이 제한되거나 로그인/차단에 걸린다.
따라서 직접 포스트 원문을 대량 수집하지 않고, 검색에 노출되는 Instagram 계열 해시태그/글귀 클러스터와
공개 커뮤니티/뉴스/설문 자료를 교차 확인했다.

사용한 관찰 축:

- `듣고 싶은 말` 설문
- 직장/가족/부부 커뮤니티에서 원하는 말
- Reddit의 `what do you need to hear right now`류 스레드
- `#위로글귀`, `#힐링글귀`, `#자존감글귀`, `#수고했어오늘도`, `#괜찮아` 계열 검색 노출

## 관찰

### 1. 사람들은 칭찬보다 먼저 "인정/수고"를 원한다

서울시민 설문에서는 `사랑해`, `수고했어`, `잘하고 있어`가 상위권이었다. 특히 20-30대 여성은
`잘하고 있어`를 가장 듣고 싶어 했고, 부모에게 듣고 싶은 말에서도 `잘하고 있어`, `괜찮아`,
`자랑스러워`가 반복됐다.

직장인 조사에서도 `수고했어`, `실력 많이 늘었네`, `믿고 맡길 수 있지`, `네가 든든해` 같은
인정/신뢰형 문장이 반복된다.

제품 해석:

- 이 욕구는 단순 "칭찬"보다 넓다.
- 핵심은 `내가 한 일을 누가 알아봐줬으면 좋겠다`이다.
- 다만 `알아줘`는 사용자가 누르는 버튼명으로는 다소 추상적이고 어색하다.

### 2. "괜찮아"는 별도 카테고리로 남겨야 한다

고교생 대상 설문에서 부모에게 듣고 싶은 말 1위는 `실수해도 괜찮아` 계열이었다.
직장인 조사에서도 `괜찮아, 실수할 수도 있지`가 높은 순위로 나타났다.

제품 해석:

- `괜찮아`는 칭찬/인정과 다르다.
- 이것은 실패, 실수, 자책을 낮추는 카테고리다.
- `칭찬해줘`나 `알아줘` 안에 넣으면 사용 순간이 흐려진다.

### 3. "내 편"은 매우 강한 정서 카테고리다

부부/관계 설문에서는 `난 늘 당신 편이야`, `수고했어`, `고마워`, `사랑해`, `덕분에 버틸 수 있었어`
계열이 반복된다. 커뮤니티에서도 힘든 사연에는 조언보다 `네 편`, `곁에 있음`, `혼자가 아님` 계열의
반응이 강하게 작동한다.

제품 해석:

- `내 편`은 응원, 위로, 인정 사이를 잇는 상위 욕구다.
- `알아줘`보다 사용자가 바로 이해하기 쉽다.
- `내 편 들어줘`는 자극적이고 앱 이름 후보와도 잘 맞지만, `진정/휴식`까지 모두 품지는 못하므로
  카테고리 중 하나로 두는 것이 낫다.

### 4. 불안/패닉 계열은 "힘내"가 아니라 "안전/반복/호흡"을 원한다

Reddit 패닉 관련 스레드에서는 동기부여보다 `지금 안전하다`, `지나갈 것이다`, `고치려 하지 않아도 된다`
같은 짧고 반복 가능한 문장이 도움이 된다는 흐름이 보인다.

제품 해석:

- `진정하자`는 필요하다.
- 다만 `진정해`는 명령처럼 들릴 수 있다.
- 버튼명은 `숨 고르자` 또는 `진정하자`가 낫고, 실제 문장은 안전/호흡 중심이어야 한다.

### 5. "쉬어도 돼"는 독립 카테고리 가치가 있다

공개 글귀/해시태그 클러스터에서 `잠시 쉬어도 돼`, `오늘 하루 수고했어`, `어서 퇴근해`, `쉬고 싶어요`
계열이 반복된다. 직장인 조사에서도 `어서 퇴근해`, `눈치 보지 말고 퇴근해`처럼 휴식 허락형 문장이 강하다.

제품 해석:

- `쉬어도 돼`는 단순 위로가 아니다.
- 죄책감, 번아웃, 과로 상태에서 작동하는 별도 버튼이다.

## `잘하고 있어` -> `알아줘`로 바뀐 이유와 재평가

바뀐 이유:

- `잘하고 있어`는 결과 문장이지, 사용자가 누르는 욕구 이름으로는 좁았다.
- `칭찬해줘`는 직관적이지만 살짝 유치하거나 부담스러운 자기요청처럼 들릴 수 있다.
- 그래서 `인정받고 싶다`, `내가 해온 걸 봐줬으면 좋겠다`는 의미로 `알아줘`를 임시 제안했다.

재평가:

- 리서치상 `알아줘`라는 표현 자체는 시장 언어가 아니다.
- 실제 반복되는 말은 `수고했어`, `잘하고 있어`, `믿는다`, `자랑스러워`, `네 편이야`, `고마워`다.
- 따라서 카테고리명은 `알아줘`보다 `인정해줘`, `믿어줘`, `내 편 들어줘` 중 하나가 더 낫다.

## 추천 카테고리 체계

내부 분류는 다음 5개가 가장 안정적이다.

| 내부 분류 | 사용자 욕구 | 짧은 버튼 후보 | 긴 버튼 후보 |
| --- | --- | --- | --- |
| 응원 | 에너지가 필요하다 | 힘 좀 줘 | 힘내라고 해줘 |
| 안정 | 불안/흥분을 낮추고 싶다 | 숨 고르자 | 진정시켜줘 |
| 휴식 | 죄책감 없이 멈추고 싶다 | 쉬어도 돼 | 쉬어도 된다고 해줘 |
| 위로 | 실수/자책을 무해화하고 싶다 | 괜찮아 | 괜찮다고 해줘 |
| 내편/인정 | 누가 내 편이 되어주고, 알아봐주길 원한다 | 내 편 | 내 편 들어줘 |

## MVP 추천 버튼

최종 추천:

1. 힘 좀 줘
2. 숨 고르자
3. 쉬어도 돼
4. 괜찮아
5. 내 편 들어줘

이유:

- `힘 좀 줘`는 `힘내자`보다 사용자의 요구가 직접적이다.
- `숨 고르자`는 `진정해`의 명령감을 줄인다.
- `쉬어도 돼`는 가장 자연스러운 결과/욕구 표현이라 그대로 둔다.
- `괜찮아`는 이미 강한 시장 언어다.
- `내 편 들어줘`는 `알아줘`보다 자극적이고, 앱명 `멘탈 한마디`와도 잘 붙는다.

대안 버튼:

1. 힘내자
2. 진정하자
3. 쉬어도 돼
4. 괜찮아
5. 인정해줘

이 대안은 더 정돈되어 있지만, 첫 추천안보다 덜 끌린다.

## 제품명과의 궁합

`멘탈 한마디`와 가장 잘 맞는 버튼 세트는 "감정명"보다 "요청형"이다.

예:

- 나 지금 `힘 좀 줘`
- 나 지금 `숨 고르자`
- 나 지금 `쉬어도 돼`
- 나 지금 `괜찮아`
- 나 지금 `내 편 들어줘`

앱이 "상담"이 아니라 "누르면 바로 나오는 말"이 되려면, 버튼은 분석적 카테고리보다
사용자가 자기 상태에서 바로 내뱉는 말에 가까워야 한다.
