# 세션 로그: 바로 듣는 한마디 MVP 기획

작성일: 2026-06-07
actor/tool: codex

## 사용자 요청

- 이름 입력/개인화 음성은 복잡하므로 제외.
- 심플하게 5개 카테고리 x 5문장으로 가자는 방향.
- `힘내자`, `진정하자`와 비슷한 카테고리 추가 검토.
- 선택한 카테고리를 정해진 시간에 알림으로 보내고, 해당 음성을 들을 수 있는 기능 구상.
- 유명 대사/드라마/영화/명언 기반 확장 가능성 유지.
- gstack, Superpowers 스킬 활용 요청.

## 결정

- MVP를 "바로 듣는 한마디 앱"으로 재정의.
- 5개 카테고리: `힘내자`, `진정하자`, `쉬어도 돼`, `괜찮아`, `잘하고 있어`.
- 각 카테고리 5문장, 총 25문장으로 시작.
- 명대사/명언은 MVP 기본이 아니라 사용자가 준비한 별도 파일 pack으로 확장.
- 알림은 자동 음성 재생이 아니라, 정해진 시간에 알림을 보내고 탭하면 앱에서 해당 카테고리 음성을 재생하는 방식으로 설계.

- 사용자가 5개 카테고리와 25문장을 승인/수정.
- 승인 후 UI를 5버튼 바로 재생 구조로 변경.
- 오디오 파일 경로 구조와 fallback TTS 연결.
- 알림 설정 UI와 플랫폼 adapter 설계.
